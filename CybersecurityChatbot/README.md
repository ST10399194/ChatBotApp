#Cybersecurity ChatBot App

A WPF-based chatbot application designed to recognize and respond to cybersecurity-related queries. 
The chatbot utilizes a simple rule-based approach to provide answers to common cybersecurity questions.

##Features
- User-friendly interface for interacting with the chatbot.
- Real-time responses to cybersecurity queries.
- Rule-based system for providing accurate answers.
- Scrollable chat history for easy reference.
- Customizable appearance with dark mode support.
